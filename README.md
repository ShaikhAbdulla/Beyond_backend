Beyond Media Gallery - Backend API
This is the robust Node.js/Express backend for the Beyond Media Gallery application. It provides secure user authentication, media management, and favorites synchronization, all containerized for consistent deployment.

Features
JWT Authentication: Secure login and signup with encrypted password storage.

Role-Based Access: Middleware-protected routes for managing user-specific artworks.

Media Management: Endpoints to fetch gallery items and toggle favorite status.

Containerized Environment: Fully configured with Docker and Docker Compose for easy setup.

Tech Stack
Runtime: Node.js

Framework: Express.js

Database: MongoDB with Mongoose ODM

Security: JSON Web Tokens (JWT) & bcrypt.js

Deployment: Docker & Render
